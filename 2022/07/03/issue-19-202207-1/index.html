SDKMAN!

A tool for managing parallel versions of multiple Software Development Kits on most Unix based systems.

You can use it to install Flink, Gradle, Groovy, Hadoop, JDK, JDK Mission Control (JMC), JMeter, Kotlin, Maven, Maven Daemon, Quarkus, sbt, Scala, Spark, Spring Boot CLI, Tomcat, Vert.x, VisualVM, etc.

For each package, SDKMAN appends its path to the Shell PATH. If you install all the packages above, the PATH will be very long.
